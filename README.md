# ezCompiler-The-simple-C-compiler
This makes compiling C files very easy. No need to put gcc "" -o "". Just a basic ezCompile "".


# How to install!

The install is fairly simple

** PLEASE INSTALL UNZIP IF NOT ARLEADY INSTALLED **

Clone the repository in your terminal by:

```bash
wget https://github.com/Madden321pop/ezCompiler-The-simple-C-compiler/archive/refs/heads/main.zip
unzip main.zip
cd ezCompiler-The-simple-C-compiler-main
chmod +x MainInstaller
./MainInstaller

```

Thats it. All so simple!

# How to use!

TO use it its seriously just

ezCompile <file.c>

Make sure to restart your terminal after installing!
